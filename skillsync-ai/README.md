# SkillSync AI 🚀

Full-stack Micro-SaaS for tracking study plans & habits, designed with premium aesthetics and smooth interactions.

## 🌟 Features
- **Premium Landing Page**: Conversion-optimized hero, features grid, pricing tiers, and testimonials.
- **JWT Authentication**: Secure login and signup flow with centralized state management.
- **Interactive Dashboard**: High-performance analytics with Recharts and real-time task tracking.
- **Study Plans**: Comprehensive CRUD for study topics with deadline management.
- **Habit Tracking**: Streak-based habit system with frequency settings (Daily/Weekly/Weekdays).
- **Advanced UI/UX**: Dark mode by default, glassmorphism, Framer Motion animations, and custom typography (Plus Jakarta Sans).

## 🛠️ Tech Stack
- **Frontend**: React, Tailwind CSS, Framer Motion, Recharts.
- **Backend**: Node.js, Express, MongoDB.
- **Auth**: JSON Web Tokens (JWT) & BcryptJS.

## 🚀 Getting Started

### Prerequisites
- Node.js installed
- MongoDB (Local or Atlas)

### Backend Setup
1. `cd server`
2. Update `.env` with your `MONGO_URI`.
3. `npm install`
4. `npm run dev` (starts on port 5000)

### Frontend Setup
1. `cd client`
2. `npm install`
3. `npm run dev` (starts on port 5173)
